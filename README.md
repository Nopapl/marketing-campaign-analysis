## Project Overview

### Description
Проект, выполненный по техническому заданию образовательной платформы. Данные для работы предоставлены образовательным центром

### Проделанная работа:
*    Предобработка данных: базовая чистка (дубликаты, пустые значения, типизация данных, ненужные атрибуты) и обработка редких значений, для улучшения качества работы ML моделей
*    Бинарная классификация с помощью RandomForestClassifier (дополнительно были проверены результаты предсказаний LogisticRegression и MLPClassifier)
*    Анализ А/В тестов: рассчет ARPU, среднего числа покупок, построение соответствующих графиков
*    Агломеративнуя кластеризация на основе KMeans и silhouette score, с последующим составлением портретов кластеров и предложением методов работы с ними
*    Построение модели склонности клиента к покупке на основе LGBMClassifier
*    Сформирован финальный отчет о проделанной работе, с заключениями и рекомендациями
